# Transaction version

All [transactions](/en/blockchain/transaction.md), with the exception of the [genesis transaction](/en/blockchain/transaction-type/genesis-transaction.md), have versions.

Transactions of the same [type](/en/blockchain/transaction-type.md), but of different versions, have different [binary formats](/en/blockchain/binary-format/transaction-binary-format.md).

When developing applications, it is recommended to use transactions of the latest versions.
